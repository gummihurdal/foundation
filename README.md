# Solidifying a Relationship — foundation.azurenexus.com

Ten practices for solidifying a relationship, each stated as a practice,
then grounded in the research it rests on, with full references.

Deliberately includes a **Limitations** section: most of the evidence is
correlational, Gottman's headline prediction rates are disputed, and the
samples skew narrow. The site says so rather than overselling.

## Files
- `index.html` — the whole site (Google Fonts only)
- `CNAME` — `foundation.azurenexus.com`
- `.github/workflows/deploy.yml` — Pages deploy

## Features
- Per-practice evidence strength tags (strong / mixed)
- 17 numbered references, linked from inline superscripts
- Print stylesheet: "Print the card" outputs the ten practices on one page,
  apparatus stripped — the practical artifact for the fridge door
